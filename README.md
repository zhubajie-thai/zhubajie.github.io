# Pork Shop Manager & eCommerce Platform

A comprehensive modern React application for managing a pork shop, including POS, Production Planning, Inventory, and Employee management, coupled with a customer-facing eCommerce website.

**Repository:** [https://github.com/zhubajie-thai/pork-shop.github.io.git](https://github.com/zhubajie-thai/pork-shop.github.io.git)

## Features

### Management Console (Staff)
- **Dashboard**: Real-time sales metrics, charts, and low stock alerts.
- **POS System**: Tablet-ready point of sale with barcode scanning, loyalty lookup, and crypto payment support.
- **Production Planner**: Calculate ingredient requirements (Chicken/Pork) based on orders.
- **Inventory Management**: Track stock, costs, and suppliers.
- **Employee Management**: Manage staff roles, salaries, and status.

### Customer Shop (Public)
- **Modern UI**: "Glassmorphism" design with high-quality visuals and animations.
- **Catalog**: Browse fresh cuts and products with category filtering.
- **Shopping Cart**: Fully functional cart with local storage persistence.
- **Authentication**: Customer login and registration with loyalty point tracking.

## Technology Stack
- **Framework**: React 19
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Charts**: Recharts
- **Scanning**: HTML5-QRCode
- **Routing**: React Router (HashRouter for static deployment)

## Deployment

This project is configured to be deployed to GitHub Pages.

1. Clone the repository:
   ```bash
   git clone https://github.com/zhubajie-thai/pork-shop.github.io.git
   ```

2. Open `index.html` in your browser or serve via a static file server.
